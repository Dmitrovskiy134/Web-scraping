# Парсер статей Habr.com

Программа для поиска статей на Habr.com по ключевым словам.

## Как работает

1. **Загружает** страницу со свежими статьями Хабра
2. **Находит** все статьи на странице  
3. **Проверяет** preview каждой статьи на наличие ключевых слов
4. **Если не находит** в preview - загружает полный текст статьи
5. **Выводит** подходящие статьи в формате: `дата – заголовок – ссылка`
6. **Обрабатывает** возможные ошибки

## Ключевые слова
- дизайн
- фото  
- web
- python
